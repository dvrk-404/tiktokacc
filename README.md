# 🎬 TikTok Hunt Tool (Yopmail)

أداة متطورة ومكتوبة بلغة **Python** لفحص وصيد حسابات تيك توك المتاحة باستخدام إيميلات **Yopmail**. تعتمد الأداة على نظام التخمين الذكي وإرسال النتائج مباشرة إلى بوت التليجرام الخاص بك.

---

## ✨ مميزات الأداة (Features)

*   ⚡ **سرعة عالية (Multi-Threading):** فحص متوازٍ لعدة إيميلات في نفس الوقت دون التأثير على أداء الجهاز.
*   🎯 **6 أنماط للتخمين (6 Hunting Modes):**
    1. تخمين بالأسماء العربية + 3 أرقام.
    2. تخمين عشوائي (5 أحرف فقط).
    3. تخمين مخلوط (4 أحرف وأرقام).
    4. تخمين بأسماء الدول والمحافظات + رقمين.
    5. تخمين بأسماء حديثة (جيل Z) + 3 أرقام.
    6. تخمين بالأسماء والمواليد.
*   🤖 **ربط تلقائي بالتليجرام:** إرسال الحسابات الشغالة (Hits) فوراً إلى حسابك عبر البوت.
*   🛡️ **منع التكرار (Anti-Dup):** نظام داخلي يمنع فحص الإيميل نفسه مرتين لتوفير الوقت والبيانات.
*   💾 **حفظ تلقائي:** يتم حفظ جميع الحسابات الشغالة في ملف نصي باسم `tiktok_hits.txt`.
